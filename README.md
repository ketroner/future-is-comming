# Future is comming ...

Krátká demo stránka věnovaná budoucnosti AI. Design je tmavý, "nerdovský" s neonovými akcenty.

Struktura:

- `index.html` — hlavní stránka
- `styles.css` — styly
- `script.js` — malá interakce

Rychlý start (Windows PowerShell):

```powershell
cd C:\www_webpage
# Otevřít lokálně (můžete použít Live Server nebo jednoduchý HTTP server):
# Python 3: python -m http.server 8000
start http://localhost:8000
```

Git & GitHub:

1. Inicializovat git a committovat:

```powershell
cd C:\www_webpage; git init; git branch -M main; git add .; git commit -m "Initial commit: responsive AI future site"
```

2. Vytvořit repo na GitHubu a pushnout — pokud máte `gh` CLI a jste přihlášeni:

```powershell
gh repo create <your-username>/<repo-name> --public --source=. --remote=origin --push
```

Nebo vytvořte repo přes web a poté:

```powershell
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

Pokud chcete, mohu udělat poslední krok (vytvořit repo a pushnout) — dejte vědět, jestli chcete, abych použil `gh` CLI nebo poskytněte vzdálenou URL.
